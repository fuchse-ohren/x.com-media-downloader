x.com Media Downloader

README

  x.com Media Downloaderはx.comの画像や動画などのメディアをブラウザ
  画面上で簡単にダウンロードするためのユーザースクリプトです．
  
  このスクリプトを有効化するとサムネイル右下とポストの共有ボタン右隣
  にダウンロードボタンが追加されます．１つのポストに複数のメディアが
  含まれている場合，サムネイル右下のボタンでは個別保存，共有ボタン右
  隣のボタンでは一括保存が選べます．
  
  ダウンロード履歴はブラウザに保存され，ダウンロード済みの場合，共有
  ボタン右隣のダウンロードボタンがチェックマークに置き換わります．
  
  多数のメディアを連続してダウンロードすると一時的にダウンロードでき
  なくなる場合がありますが，Xの仕様によるものですのでしばらく時間を
  空けてから再度ダウンロードをお試しください
